Context(문맥-앱의 흐름)
Application Context
-하나만 존재
-어플리케이션이 살아있는 동안 유지

Activity Context
-Activity 유지되는 동안에만 유지지-Activity마다 존재
-Context를 구현하고 기능을 추가한게 ->Activity
